#捞乎
前端组知识管理工具


##结构
###API
存放后端接口与操作
###common
放置 reset.css 之类
###components
组件
###entry
项目入口文件
###filters
过滤器
###mock
本地开发模拟数据
###utils
封装的工具，上传文件，日期选择器等
###views
页面视图
###vuex
放置 store ， actions， mutations， state

##ISSUE
Vue and vue-template-compiler should use 2.1.0 if you want to use the vue-mdl
